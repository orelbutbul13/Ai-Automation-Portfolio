# AI Automation Engineering Portfolio

Portfolio of automation systems developed during the TripleTen AI Automation Program.

These projects demonstrate workflow orchestration, API integrations, AI reasoning, prompt engineering, structured data pipelines, and scalable automation architecture.

---

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [AI Customer Feedback Automation](./AI-Customer-Feedback-Automation) | AI-powered workflow that analyzes customer feedback, extracts sentiment and intent, and generates brand-aligned email responses automatically. |
| 2 | [AI Expense Tracker Automation](./AI-Expense-Tracker-Automation) | Telegram-based automation that extracts financial data using AI and stores structured expense records in Google Sheets. |
| 3 | [AI Email Summarizer Automation](./AI-Email-Summarizer-Automation) | Automated email-processing system that summarizes incoming Gmail messages, applies conditional logic, and stores structured insights. |
| 4 | [AI Skincare Blog Automation](./AI-Skincare-Blog-Automation) | Weekly AI content automation system that researches, generates, refines, and structures long-form blog posts. |
| 5 | [Automated Client Provisioning System](./Automated-Client-Provisioning-System) | API-driven Google Drive automation that provisions structured client workspaces using batch HTTP requests. |
| 6 | [Weather-Based Sales & Staffing Automation](./Weather-Based-Sales-Staffing-Automation) | AI-powered decision automation that uses real-time weather data to generate proactive sales and staffing recommendations. |
