# AI Automation Engineering Portfolio

Collection of automation systems designed and implemented using workflow orchestration, API integrations, large language models, and operational automation frameworks.

These projects demonstrate scalable automation architecture, structured data pipelines, AI-assisted decision logic, workflow optimization, and multi-system operational automation.

---

## Projects

| # | Project | Description |
|---|---------|-------------|
| 1 | [AI Support Triage & Escalation System](https://github.com/orelbutbul13/Ai-Automation-Portfolio/tree/main/AI-Support-Triage-Escalation-System) | LLM-driven multi-channel automation that classifies service requests, determines urgency, logs structured records, and triggers Slack + SMS escalation using deterministic JSON routing logic. |

## Core Competencies

- **Workflow Automation:** End-to-end workflow orchestration and operational automation design.
- **API Integration:** REST APIs, webhooks, external data ingestion, and multi-system connectivity.
- **LLM Systems:** Prompt engineering, structured outputs, deterministic JSON enforcement, and AI-assisted routing logic.
- **Automation Architecture:** Scalable automation infrastructure, conditional logic systems, and workflow optimization.
- **Operational Intelligence:** Multi-channel escalation systems, automated notifications, and operational monitoring workflows.
- **Data Pipelines:** Structured data collection, transformation, logging, and process automation pipelines.
- **Business Process Optimization:** Elimination of repetitive manual processes through intelligent automation systems.
